## Project Overview
This project is an Employee Management System built using SQL. It manages employee records, job departments, salaries, payroll, qualifications, and leave details to generate meaningful business insights.

## Database Name
emp_mange_sy

## Tables Created
- JobDepartment
- SalaryBonus
- Employee
- Qualification
- Leaves
- Payroll

## Key Features
- Relational database design using primary & foreign keys
- Payroll and salary analysis
- Employee, department, and job role insights
- Leave and compensation analysis

## SQL Concepts Used
- CREATE DATABASE & TABLE
- PRIMARY KEY & FOREIGN KEY
- JOINS (INNER, LEFT)
- GROUP BY, ORDER BY
- Aggregate Functions (COUNT, SUM, AVG)
- Subqueries
- Constraints (CASCADE, SET NULL)

## Business Questions Answered
- Which departments have the highest number of employees?
- Who are the top 5 highest-paid employees?
- What is the total salary expenditure of the company?
- Which department receives the highest bonuses?
- How do employee leaves impact payroll?

This project demonstrates how SQL can be used to manage organizational data efficiently and derive actionable insights for decision-making.
